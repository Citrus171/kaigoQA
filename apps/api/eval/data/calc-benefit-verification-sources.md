calc-006	テクノロジー導入の加算	正式名は「生産性向上推進体制加算」の可能性(I)=100単位/月・(II)=10単位/月。見守り機器＋委員会設置・データ測定	低(名称要確認)
https://www.totec-mlife.jp/blog/kaigohousyuu/20250827

生産性向上推進体制加算とは？【制度の概要】
2024年度（令和6年度）の介護報酬改定で新設された「生産性向上推進体制加算（Ⅰ）（Ⅱ）」は、テクノロジー導入と業務改善の継続的な取り組みを評価する加算制度です。介護現場の生産性向上を支援することを目的とし、ICT機器や介護ロボットの導入、委員会の設置、業務改善の成果確認などが要件に含まれています。
生産性向上推進体制加算（Ⅱ）では、利用者１名あたり月10単位、 生産性向上推進体制加算（Ⅰ）では、利用者１名あたり月100単位の取得が可能です。
