## 評価軸の定義

- **relaxed（正規KPI）**: 利用者が次に取るべき行動・理解すべき主要事実を得られれば good。法令条番号・条文引用・年号・限度額・行政注記の欠落は許容。**ただし質問の核心（帰結情報）の欠落は不可**
- **strict（副軸・参考）**: gold 全要点を網羅して good（従来通り）
- factual / overreach は両軸とも厳格維持
